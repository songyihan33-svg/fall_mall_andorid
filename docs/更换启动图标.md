# 如何更换启动图标

## 当前结构

- **AndroidManifest** 中：`android:icon="@mipmap/ic_launcher"`、`android:roundIcon="@mipmap/ic_launcher_round"`
- **自适应图标**（Android 8+）由两个 drawable 组成：
  - **背景**：`res/drawable/ic_launcher_background.xml`
  - **前景**：`res/drawable/ic_launcher_foreground.xml`
- `res/mipmap-anydpi-v26/` 下的 `ic_launcher.xml`、`ic_launcher_round.xml` 引用上述两个 drawable。

---

## 方法一：用 Android Studio 生成（推荐）

1. 右键 `res` → **New** → **Image Asset**。
2. **Icon Type** 选 **Launcher Icons (Adaptive and Legacy)**。
3. **Foreground Layer**：
   - 选 **Image**：用你自己的 PNG（建议 512×512），或选 **Clip Art** 选系统图。
   - 选 **Path**：粘贴矢量 path 或从 SVG 导入。
4. **Background Layer**：选颜色或图片。
5. 点击 **Next** → **Finish**，会覆盖/生成 `ic_launcher_*` 和 drawable。

---

## 方法二：直接改 drawable

- **改前景（主图案）**：编辑 `app/src/main/res/drawable/ic_launcher_foreground.xml`，改成你的矢量图（viewport 建议 108×108）。
- **改背景**：编辑 `app/src/main/res/drawable/ic_launcher_background.xml`，可改成纯色或简单形状。

改完后无需改 Manifest，重新运行即可看到新图标。

---

## 方法三：用一张图做传统图标（非自适应）

1. 准备各分辨率 PNG（或只准备 xxxhdpi 一张）：  
   `res/mipmap-xxxhdpi/ic_launcher.png`（192×192）等。
2. 在 **Image Asset** 里选 **Launcher Icons (Legacy Only)** 并选择该图，生成 mipmap。
3. Manifest 里保持 `android:icon="@mipmap/ic_launcher"` 即可（roundIcon 可同图或单独准备）。

---

## 注意事项

- 前景图建议留出安全区：内容尽量在中间 66dp×66dp（viewport 108 下约 19–89 范围），避免被圆形/圆角裁掉。
- 背景与前景会叠加；前景可透明，背景会填满图标区域。
